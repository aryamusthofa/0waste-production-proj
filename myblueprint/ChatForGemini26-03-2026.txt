## 📋 Penjelasan Detail Project: **0waste**



### Apa itu 0waste?

**0waste** adalah platform marketplace ekonomi sirkular yang menyelamatkan **surplus makanan** (sisa makanan berkualitas) dari restoran, hotel, dan kafe, lalu menjualnya ke masyarakat dengan **harga diskon signifikan**. Visinya: mengurangi food waste di Indonesia lewat teknologi.



### Tech Stack Asli

- **React Native + Expo SDK 55** (mobile app, BUKAN Next.js — jadi koreksi dari asumsi awalmu)

- **NativeWind** (Tailwind CSS untuk React Native)

- **Supabase** (Auth, Database, Storage)

- **Google Gemini 1.5 Flash** (AI Engine)

- **TypeScript**



### Fitur-fitur Utama



| Fitur | Detail |

|-------|--------|

| **Role-Based Access** | User/Customer bisa browse & pesan. Verified Partner (restoran/hotel) bisa posting surplus makanan |

| **Sistem Verifikasi Partner** | Alur pendaftaran ketat + panel verifikasi admin internal |

| **Row Level Security (RLS)** | Keamanan data di level database Supabase |

| **Real Image Upload** | Upload foto makanan ke Supabase Storage |

| **Smart Search & Filter** | Pencarian + filter kategori (Bakery, Buffet, Beverages, dll) |

| **Real-time Stock** | Auto "Sold Out" setelah dipesan |

| **AI Assistant 2.0** | Gemini-powered: konsultan lingkungan, asisten inventaris, brand guide |

| **Developer Mode** | Tap foto profil 10x → menu verifikasi partner (internal testing) |

| **Security UI** | Show/hide password, forgot password flow |



### Evolusi Project

1. `nextjs-boilerplate-test` → boilerplate awal (Next.js)

2. `aifordevfinalproj` → development dengan AI integration

3. `0waste-production-proj` → **production version** (React Native/Expo) — Phase 4 terakhir

# 🌱 0waste Web App — Full Rebuild Plan



## Overview

Rebuild aplikasi mobile 0waste (React Native/Expo) menjadi web app menggunakan React + Vite + Tailwind CSS, dengan connect ke Supabase existing milik user.



## Fase 1: Foundation & Auth

1. **Design System** — Warna hijau/eco-friendly (#3ec976 primary), tipografi modern, responsive mobile-first layout

2. **Splash/Welcome Page** (`/`) — Hero section dengan branding 0waste, tagline "Selamatkan makanan, selamatkan bumi", CTA Login/Register

3. **Login Page** (`/login`) — Email + password, show/hide password toggle, link ke forgot password & register

4. **Register Page** (`/register`) — Form registrasi dengan pilihan role (User atau Partner), validasi form

5. **Forgot Password** (`/forgot-password`) — Input email untuk reset password

6. **Reset Password** (`/reset-password`) — Form set password baru setelah klik link email

7. **Connect Supabase** — Hubungkan ke project Supabase existing untuk Auth, Database, dan Storage



## Fase 2: Core Marketplace

8. **Home/Marketplace** (`/marketplace`) — Grid card surplus makanan, search bar, filter kategori (Bakery, Buffet, Beverages, dll), badge "Sold Out" otomatis

9. **Product Detail** (`/product/:id`) — Gambar besar, deskripsi, harga asli vs harga diskon, info partner/restoran, tombol "Beli Sekarang"

10. **Navigation** — Responsive navbar/sidebar dengan menu: Home, Orders, AI Chat, Profile

11. **Real-time Stock** — Subscribe ke perubahan stok di Supabase, update UI real-time



## Fase 3: Orders & Profile

12. **Order Flow** — Tombol beli → konfirmasi pesanan → update stok otomatis di database

13. **Order History** (`/orders`) — Daftar semua pesanan user dengan status (pending, completed, cancelled)

14. **Profile Page** (`/profile`) — Info user, edit profile, upload avatar, logout button



## Fase 4: Partner System

15. **Partner Dashboard** (`/partner`) — Dashboard khusus Verified Partner: tambah produk, upload foto ke Supabase Storage, kelola listing, lihat pesanan masuk

16. **Partner Registration Flow** — Form pendaftaran partner dengan data bisnis, menunggu verifikasi admin



## Fase 5: Admin & AI

17. **Admin Panel** (`/admin`) — Halaman verifikasi partner: daftar pending partner, approve/reject, protected route hanya untuk admin

18. **AI Chat** (`/chat`) — Chatbot AI powered by Lovable AI (menggantikan Gemini langsung): konsultan lingkungan, asisten inventaris, brand awareness 0waste. Menggunakan edge function + streaming response



## Technical Notes

- Semua halaman responsive (mobile-first karena target user mobile web)

- Auth state management via Supabase `onAuthStateChange`

- Protected routes: marketplace, orders, profile, partner, admin

- Image upload via Supabase Storage

- AI chat via Lovable AI edge function (menggantikan direct Gemini API call)
