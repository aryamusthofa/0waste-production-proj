JSON
{
  "name": "armuzz_ai_dev",
  "version": "1.0.0",
  "type": "module",
  "dependencies": {
    "@google/generative-ai": "^0.1.0",
    "dotenv": "^16.4.5"
  }
}
2. Lanjutkan Instalasi
Setelah file di atas disimpan, coba jalankan lagi perintah ini di terminal:
npm install

Ini akan mendownload library Google AI dan dotenv yang diperlukan untuk menghubungkan kodinganmu ke otak Gemini.

3. Siapkan "Brankas" API Key
Buat file baru bernama .env.

Masukkan API Key yang sudah kamu ambil dari Google AI Studio (ai.google.dev):
GEMINI_API_KEY=isi_dengan_key_kamu_disini

4. Tes Koneksi (Hello World)
Buat file bernama index.js, lalu kopas kode dasar ini untuk memastikan laptopmu sudah bisa "ngobrol" dengan Gemini:

JavaScript
import { GoogleGenerativeAI } from "@google/generative-ai";
import dotenv from "dotenv";

dotenv.config();

const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY);
const model = genAI.getGenerativeModel({ model: "gemini-1.5-flash" }); // Pakai Flash biar cepat

async function run() {
  const prompt = "Halo Gemini, saya Arya Musthofa Roja, IT Student yang lagi ngerjain project. Coba sapa saya!";
  const result = await model.generateContent(prompt);
  console.log(result.response.text());
}

run();
5. Jalankan!

Key: Content-Type

Value: application/json

JSON

Nama project

0waste AI Assistant

Siapa target pengguna chatbot Anda?

Pelanggan yang ingin mendapatkan makanan berkualitas dengan harga terjangkau, serta pemilik usaha kuliner (restoran, kafe, atau hotel) yang ingin mengelola stok makanan berlebih (daily surplus) agar tidak menjadi limbah.

Bagaimana chatbot Anda dapat membantu pengguna?

Chatbot ini membantu pengguna dengan memberikan informasi menu makanan harian yang didiskon, menjelaskan konsep leftover meal yang masih sangat layak konsumsi secara edukatif, serta memberikan panduan mengenai opsi pengiriman atau pengambilan mandiri (pick up). Chatbot ini juga dirancang untuk memperkuat identitas brand melalui gaya bahasa yang modern dan penggunaan parameter teknis seperti System Instruction dan generation configuration (Temperature) untuk menjaga akurasi respons.
